# CanRent

CanRent is a straightforward web application that aggregates listings from different real estate browsers online.  
The application displays rental listings depending on fiters and search options chosen by the user.  
We decided on implementing a minimalist design that prioritizes ease of use and quick navigation to shorten 
the time that would otherwise be required to search through many different sites.

## Features

- Browsing rental postings across various websites.
- Comparisons between saved listings.
- Filtering based on area, pricing, and ratings.
